Treasure Hunt

Create outdoor, indoor and desktop adventures by using mobile devices in Moodle

Install

Install from Moodle

You can install it directly from the Moodle plugins directory through the Site administration > Plugins > Install plugins menu. Just browse the directory and select install. (Your Moodle site need to be registered.)

Download the module

You can download it from the link below on the Moodle plugins webpage. After downloading it, unzip the content in the /mod folder of the Moodle directory

DOWNLOAD

Alternatively, you can download it from the GitHub repository.

GitHub

Install it in Moodle

Log in to your Moodle webpage as an administrator and visit the administration page. All the plugins to be updated or installed will appear, including the Treasure Hunt module. Click on the Update Moodle database button in order to proceed with the installation.

Configure the default settings

You can configure the maximum grade, the grading method and the penalties for failure in the location or failure in the answer of a user, that will appear by default when creating an instance of the activity in a course. Besides, it allows you to configure the time in seconds that it will take to unlock the edit page once the editor has left it as well as the updating period of the game screen. Take into account that the smaller the timings, the more overloads with update requests to the Moodle server.
